Instructions

This project has two parts: one about COVID-19 and the other about Monetary Policy Rate. You need to solve the questions using Ratio and Proportion concept. Sample data is provided to help you understand the project. Download the questions your group chose to work on.

COVID-19 (Work 1)

  1. Click on 'Covid 19' to view and download the questions.
  2. After downloading or previewing the questions, click on Covid 19 data files to view datasets.
  3. The data files link opens a folder with datasets named after countries (e.g., Afghanistan.xlsx).
  4. Choose your preferred country's dataset to download.
  5. Start answering the questions once you have the questions and data.
  6. Note: Afghanistan is used as an example to illustrate how to answer the questions.

Monetary Policy Rate (Work 2)

  1. Click on 'Monetary Policy Rate' to view and download the questions.
  2. Click on Monetary Policy Rate Data Files to download the dataset.
  3. The dataset contains data for a specific four-year range (e.g., 1990 to 1994).
  4. Choose the four-year range you want to work with to answer the questions.
